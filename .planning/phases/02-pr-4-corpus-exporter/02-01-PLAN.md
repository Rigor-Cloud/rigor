<objective>
Implement `rigor refine export` -- a streaming JSONL corpus exporter that reads
`~/.rigor/violations.jsonl` line-by-line, transforms each ViolationLogEntry into
a training-ready CorpusRow (one per claim), applies `--constraint` and `--since`
filters, and writes to `--out <path>` or stdout.

Purpose: Unblocks Phase 3E GEPA prompt optimization and Phase 4E Modal
discriminator training by providing a training-ready corpus from the violation
log.

Output: Extended `crates/rigor/src/cli/refine.rs` with CorpusRow, export logic,
subcommand enum, and tests. Updated `crates/rigor/src/cli/mod.rs` dispatch.

```rust
fn parse_since(s: &str) -> Result<chrono::DateTime<chrono::Utc>> {
    if let Ok(dt) = chrono::DateTime::parse_from_rfc3339(s) {
        return Ok(dt.with_timezone(&chrono::Utc));
    }
    if let Ok(d) = chrono::NaiveDate::parse_from_str(s, "%Y-%m-%d") {
        let ndt = d.and_hms_opt(0, 0, 0).context("Invalid date")?;
        return Ok(chrono::DateTime::<chrono::Utc>::from_naive_utc_and_offset(ndt, chrono::Utc));
    }
    anyhow::bail!("Expected YYYY-MM-DD or RFC3339 timestamp")
}
```

<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| Violation log file -> export process | File on local disk, owned by current user; untrusted content (malformed JSON lines) |
| --out path -> filesystem | User-provided output path; OS handles path validation |

## STRIDE Threat Register

| Threat ID | Category | Component | Disposition | Mitigation Plan |
|-----------|----------|-----------|-------------|-----------------|
| T-02-01 | Tampering | violations.jsonl malformed lines | mitigate | serde_json::from_str fails on malformed; skip line, continue (no panic) |
| T-02-02 | Tampering | --out path traversal | accept | std::fs::File::create uses OS path resolution; export is local CLI tool run by file owner; no privilege escalation vector |
| T-02-03 | Information Disclosure | Violation log content in export | accept | Export reads same file user already owns; no new disclosure surface |
| T-02-04 | Denial of Service | Unbounded violations.jsonl | mitigate | Streaming BufReader line iteration; constant memory regardless of file size (REQ-007) |
</threat_model>
