<threat_model>
## Trust Boundaries

| Boundary | Description |
|----------|-------------|
| moka TTL -> cache eviction | TTL misconfiguration could serve stale verdicts or drop valid cache entries |
| Concurrent access -> content store | Race conditions could corrupt stored content |
| Gate timeout -> auto-reject | Misconfigured timeout could auto-approve (opposite of spec) or hang forever |
| Oneshot channel -> gate decision | Channel misuse could drop decisions silently |
